Глоссарий

Определения ключевых терминов

Скин-эффект

Явление, при котором высокочастотный электрический ток вытесняется к поверхности проводника, увеличивая эффективное сопротивление и затухание сигнала.

Полное внутреннее отражение

Явление, когда свет, падающий из более плотной среды на границу с менее плотной под углом больше критического, полностью отражается обратно. Используется в оптоволокне.

Модовая дисперсия

Уширение светового импульса в многомодовом волокне из-за разного времени прохождения разных лучей. Ограничивает дальность передачи.

PoE (Power over Ethernet)

Технология передачи питания и данных по одному кабелю Ethernet. Используется для камер, точек доступа, VoIP-телефонов.

WDM (Wavelength Division Multiplexing)

Метод уплотнения, при котором по одному оптическому волокну передаётся несколько сигналов на разных длинах волн. CWDM (8-18 каналов), DWDM (40-160 каналов).

OTDR (Optical Time-Domain Reflectometer)

Прибор для диагностики оптических линий, измеряющий затухание, длину волокна и локализующий неоднородности.

NEXT (Near-End Crosstalk)

Перекрестные наводки на ближнем конце – помеха, возникающая в витой паре между соседними парами проводов.
