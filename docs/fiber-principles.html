Физика света в волноводе

Простая аналогия:

Представьте аквариум с водой. Если светить фонариком из-под воды под острым углом, луч не выйдет наружу, а полностью отразится от поверхности. Это и есть полное внутреннее отражение — основа работы оптоволокна.

Полное внутреннее отражение

Свет распространяется в сердцевине благодаря разности показателей преломления:

  • Сердцевина: n₁ = 1.46-1.48
  • Оболочка: n₂ ≈ 1.45
  • Δ = (n₁² - n₂²)/(2n₁²) ≈ 0.3-1%

Закон Снеллиуса:

n₁·sin(θ₁) = n₂·sin(θ₂)

Критический угол: θᶜ = arcsin(n₂/n₁)

При θ > θᶜ происходит полное внутреннее отражение

Числовая апертура (NA)

NA = √(n₁² - n₂²) = sin(θₘₐₓ)

где θₘₐₓ - максимальный угол ввода света

Профили показателя преломления

Тип Профиль Преимущества Недостатки
Ступенчатый Резкий переход n₁→n₂ Простота изготовления Сильная межмодовая дисперсия
Градиентный Плавное изменение n от центра Уменьшение дисперсии Сложнее в производстве

Моды распространения

Уравнение волновода:

∇²E + k₀²n²(r)E = 0

где k₀ = 2π/λ

V-число (нормированная частота)

V = (2πa/λ)·NA

где a - радиус сердцевины

Многомодовое волокно (MMF)

Диаметр сердцевины: 50/62.5 мкм

Характеристики:

  • Много путей распространения
  • NA = 0.2-0.3
  • V > 2.405
  • Число мод: N ≈ V²/2
  • Дальность: до 550 м (10 Гбит/с)
  • Применение: ЛВС, ЦОД

Одномодовое волокно (SMF)

Диаметр сердцевины: 8-10 мкм

Характеристики:

  • Только одна мода LP₀₁
  • NA = 0.1-0.14
  • V < 2.405
  • Поле: J₀(ur/a) для r ≤ a
  • Дальность: сотни км
  • Применение: магистрали

Тип ITU-T Особенности Применение
Стандартное G.652 Нулевая дисперсия при 1310 нм Общего назначения
Смещенной дисперсии G.653 Нулевая дисперсия при 1550 нм Магистрали

Дисперсия: враг скорости

Межмодовая дисперсия

Только для MMF

Δτ = L·(n₁/c)·Δ

где Δ = (n₁ - n₂)/n₁

Для градиентного профиля уменьшается в 100-1000 раз

Материальная дисперсия

Зависимость n от длины волны

Dₘ = (-λ/c)·(d²n/dλ²)

Нулевая точка: 1310 нм для SMF G.652

Волноводная дисперсия

Зависимость скорости моды от λ/a

D_w = (n₂Δ/cλ)·V·(d²(Vb)/dV²)

где b = (β/k₀ - n₂)/(n₁ - n₂)

Поляризационная модовая дисперсия (PMD)

Δτ_PMD = D_PMD·√L

где D_PMD = 0.1-0.5 пс/√км

Основное ограничение для скоростей >40 Гбит/с

Полная дисперсия

D = Dₘ + D_w

Нулевая дисперсия достигается при компенсации материальной и волноводной дисперсии

Причины PMD

Причина Эффект Компенсация
Эллиптичность сердцевины Разная скорость для ортогональных поляризаций Круговое волокно
Механические напряжения Двулучепреломление PMD-компенсаторы
Температурные изменения Изменение Δτ со временем Адаптивная компенсация

Затухание: окна прозрачности

Основные причины затухания

Рэлеевское рассеяние

α_R = A/λ⁴

A = 0.7-0.9 дБ·км⁻¹·мкм⁴

Доминирует на коротких длинах волн

Поглощение OH⁻

Пики на 950, 1240, 1383 нм

Современное "сухое" волокно: OH⁻ < 1 ppb

Устранено в современных волокнах

Поглощение в ИК-области

От колебаний кристаллической решетки

Становится значимым при λ > 1600 нм

Определяет верхнюю границу окон прозрачности

Окна прозрачности кварца

Окно Диапазон Затухание Особенности
O-band (Original) 1260-1360 нм 0.35-0.4 дБ/км Нулевая дисперсия при 1310 нм
E-band (Extended) 1360-1460 нм 0.25-0.3 дБ/км Пик поглощения OH⁻ (1383 нм)
S-band (Short) 1460-1530 нм 0.22-0.25 дБ/км Малоиспользуемое
C-band (Conventional) 1530-1565 нм 0.18-0.2 дБ/км Минимальное затухание + EDFA
L-band (Long) 1565-1625 нм 0.2-0.22 дБ/км Расширение для DWDM систем

Современное "сухое" волокно:

Содержание OH⁻ < 1 ppb. Имеет единое сверхширокое окно прозрачности 1260-1675 нм с затуханием < 0.35 дБ/км во всем диапазоне.

Нелинейные эффекты

При высоких мощностях (>10 дБм) в оптическом волокне возникают нелинейные эффекты:

Эффект Керра

n = n₀ + n₂·I

где n₂ = 2.6·10⁻²⁰ м²/Вт для кварца

Приводит к самофокусировке

Четырехволновое смешение (FWM)

Новые частоты: ω₄ = ω₁ + ω₂ - ω₃

Опасность для DWDM систем

Подавляется неравномерным шагом каналов

Вынужденное рассеяние Мандельштама-Бриллюэна (SBS)

Обратное рассеяние на акустических волнах

Порог: P_SBS ≈ 21·A_eff/g_B·L_eff

где g_B ≈ 5·10⁻¹¹ м/Вт

Вынужденное комбинационное рассеяние (SRS)

Перекачка энергии к большим длинам волн

Порог: P_SRS ≈ 16·A_eff/g_R·L_eff

где g_R ≈ 1·10⁻¹³ м/Вт

Методы борьбы с нелинейными эффектами

  • Ограничение мощности в волокне
  • Использование волокон с большой эффективной площадью
  • Применение обратной дисперсии
  • Использование когерентных технологий с цифровой компенсацией

Усилители и регенераторы

EDFA (Erbium-Doped Fiber Amplifier):

Оптический усилитель на основе волокна, легированного ионами эрбия. Усиливает сигнал в полосе 1530-1565 нм без преобразования в электричество.

Принцип действия EDFA

  1. Накачка лазером 980 нм или 1480 нм
  2. Возбуждение ионов Er³⁺
  3. Вынужденное излучение в диапазоне 1530-1565 нм
  4. Усиление сигнала без преобразования в электричество

Измерения и тестирование

Ключевые параметры для измерения

Затухание

OTDR (Optical Time Domain Reflectometer)

Точность: ±0.05 дБ

Дисперсия

Метод фазового сдвига

Точность: ±0.1 пс/нм·км

PMD

Метод интерферометрии

Точность: ±0.01 пс/√км

BER

Bit Error Rate

Требование: <10⁻¹²

Характеристики OTDR

Параметр Значение Примечание
Длина волны 1310/1550/1625 нм Современные модели - все три
Динамический диапазон 30-45 дБ Чем больше, тем лучше
Мертвые зоны <10 м (событийная)
<30 м (затухания)		 Определяет разрешающую способность
Дальность действия до 250 км Для магистральных линий

Эксплуатационные требования

  • Минимальный радиус изгиба: 20×диаметр кабеля
  • Температурный диапазон: -40°C...+70°C
  • Срок службы: 25 лет
  • Сопротивление растяжению: ≥2000 Н
  • Устойчивость к ударам: 30 g, 11 мс

Важность тестирования:

Качественное тестирование оптических линий — залог их надежной работы на протяжении всего срока службы. Современные OTDR позволяют не только измерять параметры, но и создавать цифровые паспорта линий для последующего мониторинга.
